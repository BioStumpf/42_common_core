# Projects
The Common Core is organized into milestones, seven in total, numbered from 0 (as 42 is a coding school). Some milestones include multiple projects grouped together, allowing to choose one to complete.

## Circle -1: The Picine
Although not technically part of the Common Core, every student must complete this intensive bootcamp. It introduces the very basics of programming, serving as the first coding experience for many students.

## Circle 0
**libft ‚úÖ 125/100**   
A perfect follow-up to the Piscine. This project focuses on creating a custom C library, reimplementing many standard C functions along with some additional utility functions.

## Circle 1
**get_next_line ‚úÖ 125/100**   
This project teaches how to handle file input in C. The task is to create a function that reads from one or more files line by line, based on a specified BUFFER_SIZE.

**Born2beroot ‚úÖ 100/100**   
An introduction to virtual machines and system administration. In this project, a Linux server is set up and configured inside VirtualBox.

**ft_printf ‚úÖ 100/100**   
Every programmer starts with a ‚ÄúHello, World!‚Äù program, but this project goes further. A simplified version of the C printf function is recreated, demonstrating how variadic functions work in C.
